쉽고 빠른 알메이트-그리드
그리드의 주요 기능
  • 정렬, 이동, 셀병합, 입력, 필터링, 검색, 엑셀연동등 핵심기능을 제공.
  • 컬럼 및 로우(행), 셀에 대한 속성제어 기능제공.
  • 웹상에서 입력 및 편집/수정 기능 제공.
  • 병합(행 및 열병합)기능 및 자동병합기능 제공.
  • 자동 행높이 조절기능 제공.
  • 계층형 자료 표현(보이기/숨기기)기능 제공.
  • 엑셀과의 연동 : 엑셀 import (csv, xls, xlsx 지원), 엑셀 export (xls, xlsx 지원), 엑셀과 그리드간의 Copy & Paste 기능제공.
  • 자동합산 (소계,전체합계,합산라벨변경등)기능제공.
  • 선택제어 (한행, 여러행, 한셀, 여러셀선택)기능제공.
  • 각종 랜더러(CheckBox, Icon, Image, Spark Chart 등)제공.
  • 헤더 및 셀내에 html 표시기능제공.
  • 행의 추가/수정/삭제 여부 표시 컬럼 제공.
  • 푸터(Footer)기능제공.
  • Lazy Loading 기능제공.
  • 웹접근성 준수.
그리드의 레이아웃 및 데이터 연동
  • rMate Grid의 생성은 레이아웃(LayOut)과 데이터(Data)가 필요합니다.
  • 레이아웃은 그리드의 컬럼이나, 객체의 ID, 속성, 스타일등을 정의하기 위한 XML형식의 파일 혹은 String입니다.
  • 데이터는 JSON 형식과 배열형식, XML 형식, CSV(comma-separated values) 와 TSV(tab-separated values)형식을 모두 지원하며, 파일(URL)로 전달할 수도 있고, String으로 전달할 수 있습니다.
